# YouTube Transcript Viewer - React Frontend

Modern React frontend for the YouTube Transcript Viewer application.

## Setup

1. Install dependencies:
```bash
cd frontend
npm install
```

2. Start the development server:
```bash
npm start
```

The app will open at `http://localhost:3000`

## API Server

Make sure the Flask API server is running:

```bash
# From the project root
uv run python -m goodanalysis.main api
```

The API server runs on `http://127.0.0.1:5000` by default.
